Roomigo is a web-based platform that helps college students find off-campus housing and compatible roommates. Students can create profiles, post listings, search rooms by filters, and chat through a built-in messaging system. The site is fully connected to an Oracle SQL database, allowing listings, profiles, and student info to update automatically in real time.

**Key Features**
Student registration, login, and profile creation
Post, edit, and delete room listings
Search and filter rooms by rent, location, and lifestyle
Built-in messaging between students
Dynamic homepage showing real-time listings and student profiles
Routing that loads individual profile pages with preferences + listings

**Backend & Database**
Oracle SQL database with Student, Profile, Listing, Message tables
Real-time synchronization between database and frontend
Automatic rendering of newly added students, profiles, and rooms
Proper use of primary keys, foreign keys, and constraints
